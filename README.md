# AI 旅行规划师 — Hermes Agent Skill

基于 Hermes Agent 框架的全生命周期智能旅行助手。从出发前规划、旅途中实时调整，到旅行后生成手账回忆，覆盖一趟旅行的完整生命周期。

## 核心功能

### 出发前：引导式行程规划
- **7步信息采集**：城市、出发日期、出行方式、天数、预算、同行人、偏好
- **天气感知**：15天内出发自动查询真实天气；极端天气主动建议改期；15天外生成晴雨双方案
- **交通查询**：高铁/飞机支持班次和票价搜索（v1.4.0）
- **联网搜索**：实时搜索公开旅游攻略 + 高赞抖音/小红书视频链接
- **476个中国城市**全覆盖，200个景点别名映射（说"喀纳斯"自动匹配"阿勒泰地区"）

### 旅途中：实时动态调整（迭代1）
7种触发词，一句话调整后续行程：
"累了"/"下雨了"/"想吃X"/"不想去X了"/"还有时间"/"太贵了"/"改一下明天"

### 长期记忆（迭代2）
- 每次旅行后自动学习偏好，下次"老样子"跳过重复输入
- 6维度用户画像：目的地偏好、活动偏好、节奏、忌口、预算风格、同行模式

### 旅行后：手账生成（迭代3）
- 行程最后一天主动提醒："要存手账吗？"
- 5步采集：实际花费、高光时刻、照片与故事、一句话总结
- 输出 Markdown 手账，保存到 journals/ 目录
- 每张照片配背后故事，不用冷冰冰的星级评分

### 平台适配
- **飞书/Lark**：精简提问（每问不超过15字），最终仅输出卡片
- **桌面/CLI**：自然对话，完整 Markdown 输出

## 使用方法

### 桌面端
在 Hermes 对话中直接说"我想去XX旅游"，Agent 按步骤引导完成规划。

### 飞书端
1. 配置飞书网关：hermes gateway setup
2. 将 skill/ 目录复制到 Hermes 技能目录
3. 飞书群内 @机器人 发送"想去XX"

### 手动加载
/skill travel-planner
